1. 密码哈希（必须与论坛一致）

论坛与 BSS 本地 AuthRepository.hashPassword 相同：SHA-256 UTF-8 明文 → 小写 64 位 hex，无 salt。

import hashlib

def hash_password(password: str) -> str:
    return hashlib.sha256(password.encode("utf-8")).hexdigest()

# hash_password("test") == "9f86d081884c7d659a2feaa0c55ad015a3bf4f1b2b0b822cd15d6c15b0f00a08"
# hash_password("secret123") == "ecd71870d1963316a97e3ac3408c9835ad8cf0f3c1bc703527c30265534f75ae"

BSS 注册/登录时传明文密码即可，论坛端自行哈希存储。

2. ID 映射关系

bss SQLite users.id        → 论坛 users.bss_user_id（首次 sync 时自动绑定）
bss SQLite documents.id    → 论坛 posts.bss_document_id（UNIQUE，防重复同步）
bss SQLite folders.id      → 论坛 posts.bss_folder_id
论坛 users.id              → 论坛 posts.author_id（发帖人）

注意：论坛 users.id 与 bss users.id 是两套独立自增 ID，通过 bss_user_id 关联。

5. 帖子同步 API（BSS 核心对接）

POST /api/posts/sync（需 Bearer）

由 BSS 将 documents 记录同步为社区帖子。社区 Web 端不提供发帖入口，发帖只在 BSS。

// 请求
{
  "bssUserId": 3,
  "bssDocumentId": 42,
  "bssFolderId": 7,
  "bssFolderName": "工作笔记",
  "title": "今日开发日志",
  "content": "# 标题\n\nMarkdown 正文...",
  "bssDocumentUpdatedAt": "2026-07-02T15:30:00.000"
}

┌──────────────────────┬─────────────────────────────────┬──────┐
│ 字段                 │ 来源                            │ 必填 │
├──────────────────────┼─────────────────────────────────┼──────┤
│ bssUserId            │ documents.user_id               │ 是   │
├──────────────────────┼─────────────────────────────────┼──────┤
│ bssDocumentId        │ documents.id                    │ 是   │
├──────────────────────┼─────────────────────────────────┼──────┤
│ bssFolderId          │ documents.folder_id             │ 是   │
├──────────────────────┼─────────────────────────────────┼──────┤
│ bssFolderName        │ folders.name（BSS 查库后附带）  │ 否   │
├──────────────────────┼─────────────────────────────────┼──────┤
│ title                │ documents.title                 │ 是   │
├──────────────────────┼─────────────────────────────────┼──────┤
│ content              │ documents.content               │ 是   │
├──────────────────────┼─────────────────────────────────┼──────┤
│ bssDocumentUpdatedAt │ documents.updated_at（ISO8601） │ 是   │
└──────────────────────┴─────────────────────────────────┴──────┘

业务逻辑（论坛端已实现，BSS 需理解）：

1. 首次 sync：若论坛账号 bss_user_id 为 NULL，自动绑定请求中的 bssUserId
2. 已绑定且 bssUserId 不一致 → 403 bss用户ID与当前账号不匹配
3. bssUserId 被其他论坛账号占用 → 409 该bss用户ID已绑定其他社区账号
4. 同一 bssDocumentId 已存在且 bssDocumentUpdatedAt 更新 → UPDATE 标题/正文
5. 已存在且无更新 → 幂等返回现有帖子
6. 尝试更新他人帖子 → 403 只能更新自己的帖子

// 成功 201
{
  "id": 1,
  "authorId": 1,
  "authorUsername": "alice",
  "title": "今日开发日志",
  "content": "# 标题\n\nMarkdown 正文...",
  "bssUserId": 3,
  "bssDocumentId": 42,
  "bssFolderId": 7,
  "bssFolderName": "工作笔记",
  "bssDocumentUpdatedAt": "2026-07-02T15:30:00.000",
  "createdAt": "2026-07-02T16:00:00.000",
  "updatedAt": "2026-07-02T16:00:00.000"
}

┌────────┬─────────────────────────────────────────────────┐
│ 状态码 │ detail                                          │
├────────┼─────────────────────────────────────────────────┤
│ 401    │ 无效令牌                                        │
├────────┼─────────────────────────────────────────────────┤
│ 403    │ bss用户ID与当前账号不匹配 / 只能更新自己的帖子  │
├────────┼─────────────────────────────────────────────────┤
│ 409    │ 该bss用户ID已绑定其他社区账号 / 该bss文档已同步 │
└────────┴─────────────────────────────────────────────────┘

───

BSS 端同步流程（必须实现）

1. 用户对某 documents 执行「同步到社区」
2. 读取 title、content、user_id、folder_id，JOIN folders 取 name
3. 带 JWT 调 POST /api/posts/sync
4. 成功后将 documents.synced_to_community = 1
5. 若文档 updated_at 变更，可再次 sync 更新社区帖子（幂等）

───

DELETE /api/posts/{postId}（可选，需 Bearer，仅作者）

• 成功：204 无响应体
• 403：只能删除自己的帖子
• 404：帖子不存在

删除后 BSS 自行将本地 synced_to_community 重置为 0（论坛不会回调 BSS）。

9. BSS 端必须实现的工作清单

9.1 CloudAuthApi HTTP 客户端
• 基址：https://forum.mahoer.space
